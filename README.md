Developed a scalable Personal Finance Intelligence Platform using Spring Boot and PostgreSQL, implementing JWT-based authentication and role-based authorization.
Enhanced system performance using Redis caching with proper cache invalidation strategies, reducing API response time from ~200ms to ~5ms. Optimized database queries using indexing,
DTO projections, and resolved N+1 problems using join strategies. Built dynamic filtering APIs using JPA Specification, combined with pagination and validated sorting. 
rate limiting to prevent API abuse and conducted performance benchmarking using SQL analysis and execution time logging.
